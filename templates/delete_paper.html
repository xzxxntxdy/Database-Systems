删除论文（含有事务应用的删除操作）

使用数据库事务删除论文及其关联的作者关系、关键词关系、投稿记录、审稿记录和引用关系。如果任何一步失败，事务将回滚。

删除论文
事务说明
删除涉及的表（6张）：
  1. review — 审稿记录
  2. submission — 投稿记录
  3. paper_author — 论文作者关系
  4. paper_keyword — 论文关键词关系
  5. citation — 论文引用关系
  6. paper — 论文本身
表连接字段：
  • review.submission_id = submission.submission_id
  • paper.paper_id = paper_author.paper_id
  • paper.paper_id = paper_keyword.paper_id
  • paper.paper_id = submission.paper_id
  • paper.paper_id = citation.citing_paper_id
  • paper.paper_id = citation.cited_paper_id
删除条件：

paper.paper_id = ?

事务控制：

Python 代码中使用 conn.commit() 提交事务，使用 conn.rollback() 回滚事务。
